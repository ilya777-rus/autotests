## Запуск тестов

В репозитории предусмотрены три скрипта для запуска автоматических тестов:

### Команды

Запуск 1 скри:

```bash
pytest -s -v --tb=line -m scen1 tests.py 
```
Запуск 2 сценария:

```bash
pytest -s -v --tb=line -m scen2 tests.py 
```
Запуск 3 сценария:

```bash
pytest -s -v --tb=line -m scen3 tests.py 
```
